HTML

CSS

Para estilizar o site a linguagem CSS permite que o site possa ficar do seu estilo e também ganhe mais vida.

JavaScript

Java Script permite que o site fique dinâmico e possa fazer mais dos mais variados tipos de animações e funções.

PHP

Com PHP o site fica dinâmico e amigável além de permitir que ele possa por exemplo mandar e-mails, conectar-se ao banco de dados dentre outros.

Responsivo

Ter um site responsivo é de grande importância para que seu site possa ver visualizado por vários dispositivos de diferentes tamanhos de tela.
